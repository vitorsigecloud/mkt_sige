Prestação de Serviços

O controle de serviços prestados pode ser realizado com tranquilidade no SIGE Cloud. De fácil utilização, é adequado a diversos tipos de serviços. Possui ainda integração automática com o estoque e o financeiro, bem como com o módulo fiscal para a geração de Notas Fiscais de Serviço (NFS-e) para mais de XXXXXX prefeituras. Para cada ordem de serviço criada, o SIGE Cloud permite que você imprima diversos documentos. Além do espelho da OS você pode imprimir recibos, ordens externas, além de enviar os documentos por e-mail.

Emissão de Boletos

Nossa ferramenta para gestão de boletos é de fácil utilização, e automatiza o procedimento de cobranças dentro do sistema, integrando as informações diretamente ao fluxo de caixa. Os boletos podem ser gerados diretamente do SIGE Lite!

    
 

Dezenas de Relatórios

O SIGE Cloud disponibiliza diversos relatórios com base nas informações cadastradas no sistema. São mais de 100 relatórios pré-configurados, além da possibilidade de personalizar relatórios da forma como gostaria de visualizar as informações. Além disso, você sempre terá filtros para buscar informações específicas e a possibilidade de exportação dos dados para o seu computador em Excel ou PDF.

Multiusuários

O SIGE Cloud permite que você cadastre outros usuários para acessar a mesma base de dados. Ou seja, todos os seus colaboradores acessam o sistema com credenciais de acesso próprias, para manter a rastreabilidade dos dados. E funciona perfeitamente com o SIGE Lite. Você pode ter diversos caixas rodando o SIGE Lite e armazenando as informações na mesma base de dados! No cadastro de usuários você pode diferenciar as permissões do usuário no SIGE Cloud, ou seja, quais são as ferramentas que o usuário poderá consultar, editar ou novos cadastros que pode realizar.

    
 

Backup

Contamos com uma rotina de backups diários, que realiza uma cópia de segurança dos dados a cada 24 horas. Sempre que for necessário, você poderá solicitar estes dados e eles serão restabelecidos no sistema. Além disso, disponibilizamos uma tela onde você pode realizar o backup dos dados sempre que julgar necessário. Esta cópia de segurança será baixada para o seu computador, onde você poderá armazenar no local em que julgar mais seguro.
